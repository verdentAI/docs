Complete keyboard shortcut reference for Verdent commands and VS Code integration.

<Info>
VS Code keyboard shortcuts can be customized through VS Code settings. Default shortcuts listed below.
</Info>

---

## Primary Shortcuts

| Action | Windows/Linux | macOS |
|--------|---------------|-------|
| Open Verdent | `Ctrl+L` | `Cmd+L` |
| Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Open Settings | `Ctrl+,` | `Cmd+,` |
| Reload Window | `Ctrl+Shift+P` → "Reload Window" | `Cmd+Shift+P` → "Reload Window" |

---

## VS Code Integration

Verdent works with standard VS Code shortcuts:

| Action | Windows/Linux | macOS | Description |
|--------|---------------|-------|-------------|
| Save File | `Ctrl+S` | `Cmd+S` | Save after Verdent modifications |
| Find | `Ctrl+F` | `Cmd+F` | Search within file |
| Go to File | `Ctrl+P` | `Cmd+P` | Quick file navigation |
| Toggle Terminal | `` Ctrl+` `` | `` Cmd+` `` | View bash command outputs |
| Extensions Panel | `Ctrl+Shift+X` | `Cmd+Shift+X` | Manage Verdent extension |
