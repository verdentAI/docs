## Chat Controls

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Send Message | `Enter` | `Enter` |
| New Line | `Shift+Enter` | `Shift+Enter` |
| Switch Chat Mode | `Cmd+.` | `Ctrl+.` |
| Switch Model | `Cmd+/` | `Ctrl+/` |
