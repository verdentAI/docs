## Getting Help

### Official Support

**Feedback:**

- Open Verdent in VS Code
- Click a **Feedback **in **Top Bar**
- Describe your issue or upload a screenshot, then just hit send

You can also provide feedback by email, but be sure to include more details:

**Email Support:**

- **Email:** [support@verdent.ai](mailto:support@verdent.ai)
- **Response time:** Typically within 24-48 hours
- **Best for:** Technical issues, billing questions, account problems

**When contacting support, include:**

1. Verdent version (from Extensions panel)
2. VS Code version (`Help â†’ About`)
3. Operating system and version
4. Error messages (exact text or screenshots)
5. Steps to reproduce the issue
6. Expected vs actual behavior

## Pricing & Subscriptions

Verdent uses a credit-based pricing model with monthly subscriptions ($19-$179) and flexible top-ups. All plans include access to frontier models (Claude Sonnet 4.5, GPT-5, GPT-5-Codex).

**Pricing tiers:**

- **Free Trial:** \$0 (100 credits for 7 days)
- **Starter:** $19/month | **Pro:** $59/month | **Max:** \$179/month

<Tip>
  Credits are shared across all Verdent products (VS Code extension, desktop app), enabling seamless workflow integration between platforms.
</Tip>

**Manage subscription:** Contact [support@verdent.ai](mailto:support@verdent.ai) or access Stripe account settings.

## Release Notes

### Staying Updated

**Check for updates:**

- VS Code automatically notifies of extension updates
- View release notes in VS Code marketplace listing

**What's included in release notes:**

- New features and improvements
- Bug fixes and performance enhancements
- Breaking changes and migration guides
- Deprecation notices
