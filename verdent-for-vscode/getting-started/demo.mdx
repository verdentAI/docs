# Mintlify Styling Demo

This page demonstrates various Mintlify components and styling options available for documentation.

## Typography

### Headings

Headings from H1 through H6 are available. Use H1 for page titles (defined in frontmatter), H2 for main sections, and H3-H6 for subsections.

### Text Formatting

You can use **bold text**, *italic text*, ~~strikethrough~~, and `inline code` for emphasis.

> This is a blockquote. Use it for important callouts or quotes.

## Accordion

<Accordion title="What is an accordion?">
  An accordion is a collapsible section that can hide or reveal content. It's useful for FAQ sections or detailed information that doesn't need to be immediately visible.
</Accordion>

<Accordion title="When should I use accordions?">
  Use accordions when you have:
  - Detailed explanations that might overwhelm new users
  - FAQ sections
  - Optional advanced information
  - Long lists of configurations or options
</Accordion>

## Steps

<Steps>
  <Step title="Install Verdent">
    Download and install Verdent from the VS Code marketplace or use the command line installer.
  </Step>
  <Step title="Configure API Key">
    Set up your API key in the Verdent settings to authenticate with the service.

    ```json
    {
      "verdent.apiKey": "your-api-key-here"
    }
    ```
  </Step>
  <Step title="Start Using Verdent">
    Open any code file and start using Verdent's AI-powered features by pressing `Ctrl+Shift+V`.
  </Step>
</Steps>

## ParamField

## Summary

This page showcases the main Mintlify components available for documentation:

- **Typography**: Headings, bold, italic, code

- **Interactive**: Tabs, Accordions, Steps

- **Reference**: Tables, ParamField, ResponseField
