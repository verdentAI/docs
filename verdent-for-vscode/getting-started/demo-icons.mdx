## Best Practices

<Check>
**Do:**
- Use consistent icons across similar content types
- Match icon to content purpose (rocket for quick starts, book for docs)
- Use brand icons with `iconType="brands"` for technology logos
- Keep icon usage minimal to avoid visual clutter
</Check>

<Warning>
**Don't:**
- Mix too many different icon styles on the same page
- Use obscure icons that users won't recognize
- Overuse icons - text alone is often clearer
- Use icons without clear meaning or context
</Warning>
