<Note>
Eco Mode is currently available only in Verdent Desktop. It is not available in Verdent for VS Code.
</Note>

## What Eco Mode Is

Eco Mode lets you continue using Verdent after your credits run out.

In Eco Mode:

- You can keep working without purchasing more credits first.
- Only selected low-cost models are available.
- Usage is managed automatically to help keep access fair.

## When To Use Eco Mode

You can enable Eco Mode:

- When your credits are exhausted.
- At any time, even if you still have credits.

## How It Works

Eco Mode uses a separate usage system from credits.

- Only selected models are available in Eco Mode.
- Usage is subject to built-in limits.
- If limits are reached, requests pause until availability is restored.

## Model Availability

In Eco Mode, only Eco models are available. Other models are temporarily unavailable.

The following features are not supported in Eco Mode:

- Smart Suggestions
- Automatic compression

## Limits and Availability

Eco Mode is designed to provide continued access while keeping usage balanced.

- Usage may be temporarily limited during periods of high activity.
- Access restores automatically over time.
- No action is required from you.

## Switching Modes

### Enable Eco Mode

You can enable Eco Mode:

- From the model selector
- From the dialog shown when your credits run out

### Disable Eco Mode

- You can exit Eco Mode at any time.
- If you have no credits available, requests will not continue after you exit.

## Requirements

- Eco Mode is available in Verdent Desktop with an active subscription.
